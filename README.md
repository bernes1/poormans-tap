# poormans-tap
this is a cooking book on making a poormans network tap



gear needed for this is following: 
 - computer with 2 NICs preferably manfactured by intel.
 - 1 switch that supports Port-mirroring in this case Prosafe Plus Gs105ev2 as mentioned [here](https://wiki.wireshark.org/CaptureSetup/Ethernet#capture-using-a-monitor-mode-of-the-switch)
 - linux VM or run linux on the host
