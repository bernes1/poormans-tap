# poormans-tap
this is a cooking book on making a poormans network tap



gear needed for this is following: 
 - computer with 2 NICs preferably manfactured by intel.
 - 1 switch that supports Port-mirroring in this case Prosafe Plus Gs105ev2 similar as mentioned [here](https://wiki.wireshark.org/CaptureSetup/Ethernet#capture-using-a-monitor-mode-of-the-switch)
 - linux VM or run linux on the host



# steps

1. setup a linux machine and configure the network interfaces
2. intstall wireshark and disable ipv6 and mdns on the listening interface connected to the switch
3. WIP
